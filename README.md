# scala-web-millbuild-spring-thyme-hello-world

## Description
A springboot web app with thymeleaf support.

## Tech stack
- scala
- millbuild
  - springboot
  - thymeleaf
  - bootstrap
  - jquery
  - datatable

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
